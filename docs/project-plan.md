# Project Plan — HireTrack

## Project Goal

Build a full-stack job portal where:

- candidates can search and apply for jobs
- recruiters can post jobs and manage applicants
- admins can oversee platform activity

The goal is to create a resume-grade full-stack application that demonstrates backend engineering with Java and Spring Boot, frontend development with React, authentication, database design, role-based access, and deployment readiness.

---

## Primary Roles

### 1. Admin
Responsibilities:
- manage platform users
- monitor jobs and applications
- control recruiter access
- review system-wide data

### 2. Recruiter
Responsibilities:
- create jobs
- update jobs
- review candidates
- change application statuses

### 3. Candidate
Responsibilities:
- create account
- browse jobs
- apply to jobs
- upload resume
- track application status
